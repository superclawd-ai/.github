## Hey, we're SuperClawd 🦀🎧

**Claude Code, supercharged — your team's standards, every session.**

AI is changing how we build software. But without guidance, Claude Code reinvents the wheel every session — different patterns, forgotten conventions, inconsistent output.

SuperClawd loads your team's **skills, agents, commands, workflows, and live directives** into every Claude Code session, on demand — so the AI follows your rules automatically. No copy-pasting context, no reminding it how your team works.

- 🧠 **Your skills, on tap** — applied automatically when they're relevant
- 🤖 **Your agents** — available as Claude Code subagents
- ⚡ **Commands & workflows** — your reusable processes, ready to run
- 📏 **Always-on rules** — your standards stay in effect for the whole session

**One command. Your standards, every session.**

```bash
npm install -g superclawd
superclawd login       # sign in via your browser (one time)
superclawd             # launch Claude Code, supercharged
```
